# 📚 UBC Engineering Physics Reference Library

Welcome to my central repository of reference sheets, formula pages, and course notes compiled throughout my **Engineering Physics** degree at the **University of British Columbia**. 

I built this library to have quick, remote access to my technical materials from anywhere. If you are an ENPH student, feel free to browse these resources for your own study or use this structure as a baseline for organizing your own academic library.
